# TU Wien: Systemnahes Programmieren - Exercises

This repository contains a collection of laboratory exercises for the **Systemnahes Programmieren** (Systems Programming) course at TU Wien. Each directory corresponds to a specific assignment, covering various aspects of low-level programming, C++, and system-level concepts.

## 📂 Repository Structure

- **`1/`**: Exercise 1
- **`2/`**: Exercise 2
- **`3/`**: Exercise 3
- **`4/`**: Exercise 4 (2D Geometry & SVG Rendering)
- **`LICENSE`**: MIT License

## 🛠️ Requirements & Building

The projects are built using **CMake**. To compile and run the exercises, ensure you have a C++ compiler and CMake installed.

### Build Instructions

1.  Navigate to the exercise directory:
    ```bash
    cd 4
    ```
2.  Initialize submodules (if required):
    ```bash
    git submodule update --init --recursive
    ```
3.  Configure and build using CMake:
    ```bash
    cmake -B build
    cmake --build build
    ```
